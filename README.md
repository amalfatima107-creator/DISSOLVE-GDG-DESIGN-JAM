# DISSOLVE™

*A fictional sublingual strip. Dissolves in seconds. Like your sense of self.*

## About
DISSOLVE™ is a landing page for a fictional health product. The twist? Instead of listing side effects, you experience them — a creature runs across your screen, Farquaad judges you, the page gaslights you.

## Features
- Rotating, glitching sublingual strip
- Animated creature that runs faster each sighting
- Lord Farquaad appearing uninvited
- Gaslighting modal (changes on 2nd open)
- Broken timer, mutating title, drifting cursor
- Footer year glitch (2026 -> 1504 -> 3077)

## Built With
- HTML/CSS/JavaScript
- SVG animations (creature)
- CSS keyframes (strip, cursor, glitch)

## Submission for
GDGoC Web Design Jam - Theme: "Health product with absurd side effects"
